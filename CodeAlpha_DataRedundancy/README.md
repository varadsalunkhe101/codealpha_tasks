# Cloud Data Redundancy Removal System

## About the Project

Developed as part of the CodeAlpha Cloud Computing Internship.

This project is a Flask-based web application that detects and prevents duplicate data entries using email and mobile number validation. The system maintains duplicate logs, provides dashboard analytics, and helps optimize data storage efficiency.

## Features

* Add Records
* View Records
* Search Records
* Delete Records
* Duplicate Email Detection
* Duplicate Mobile Number Detection
* Duplicate Logs
* Dashboard Analytics
* Responsive User Interface

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* Bootstrap 5

## Cloud Computing Concepts

* Data Redundancy Removal
* Storage Optimization
* Data Integrity
* Centralized Data Management
* Cloud-Ready Application Architecture
